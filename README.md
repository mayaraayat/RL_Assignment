# Reinforcement Learning in Text Flappy Bird: A Comparison of First-visit Monte Carlo Control and Sarsa(λ)

### Author: Mayara AYAT

## Description
We study reinforcement learning in the Text Flappy Bird environment, where
the objective is to maximize survival time using a discrete state representation
and two actions: do nothing and flap. We compare First-visit Monte Carlo control vs Sarsa(λ) (Sutton & Barto, Section 12.7). The comparison focuses on
performance, value functions, parameter sensitivity, and generalization.

## Repository Contents
This repository includes the report and the code notebook for the project. The notebook contains the implementation of both algorithms, the training process, and the evaluation of their performance. The report provides a detailed analysis of the results, including visualizations of the value functions and discussions on parameter sensitivity and generalization capabilities.
